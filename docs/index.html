Get Started

Deploy in Minutes

From zero to a running SOC competition in five commands.

1

Clone the Repository

Get the full project with scenarios, generators, and Docker configs.

git clone https://github.com/mrhackerbot/WazuhBOTS.git && cd WazuhBOTS
2

Generate Datasets

Create deterministic alert datasets for all four attack scenarios.

python3 scripts/generate_datasets.py --all
3

Launch the Stack

Bring up Wazuh Manager, Indexer, Dashboard, CTFd, and Nginx.

cp .env.example .env # Edit passwords first docker compose up -d
4

Ingest Alerts

Load pre-generated alerts into the Wazuh Indexer with proper mappings.

python3 scripts/ingest_datasets.py --all
5

Load Challenges into CTFd

Import all 150 challenges with flags, hints, and scoring into CTFd.

python3 scripts/generate_flags.py
